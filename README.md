# 🚀 OnboardAI — Enterprise AI Onboarding SaaS Platform

OnboardAI is an enterprise-style AI onboarding platform that generates structured, multi-domain learning roadmaps for employees, students, or developers.

It simulates how modern companies use AI to design personalized onboarding journeys.

This project is built as a hackathon-ready SaaS prototype using a beginner-friendly tech stack.

---

## 🧠 Problem Statement

Modern onboarding processes are:

- Static and non-personalized
- Not domain-adaptive
- Poorly structured for skill development
- Inefficient for enterprise scale

OnboardAI solves this by generating dynamic AI-driven onboarding roadmaps.

---

## ✨ Key Features

### ✅ Multi-Domain AI Onboarding

Users can select multiple domains:

- Fintech
- Artificial Intelligence
- Cloud Computing
- Cybersecurity
- Healthcare

The system generates a context-aware combined learning roadmap.

### ✅ Hierarchical Learning Structure

Roadmaps follow an enterprise learning architecture:

Week → Module → Topic → Task

This mirrors real corporate training programs.

### ✅ Enterprise SaaS UI

- Modern dark gradient design
- Glassmorphism cards
- Collapsible roadmap tree
- Dynamic roadmap rendering
- Smooth user experience

### ✅ AI-Simulated Roadmap Engine

- Backend simulates intelligent planning
- Modular architecture ready for GPT integration
- Context-aware domain blending

### ✅ Hackathon-Ready Architecture

- Single file deployable app
- Zero external database
- Local execution ready
- Beginner readable clean code

## ⚙️ How to Run Locally

1. Install dependencies

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
